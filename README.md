Terrarium engine
==

A 2-d sandbox game engine written in pygame

### Installing
You will need to install python 3, pygame and, optionally for maketilemap.py script, PIL

### Features
- Python modding API
    - Entities
    - Blocks
    - Items
    - Parallax
    - Biomes and ores generation
- JSON modding API
    - Blocks
